Bee Murdering Game

This is a simple game to show how to use JavaScript objects.

Rules

  • There are 3 types of bees. Queen, Worker and Drone. Each have a different lifespan.
  • When you hit a bee, they lose some years and are closer to dying. The impact a hit has is different per bee.
  • When all the bees die, the game is reset.
  • When the queen bee dies, all bees die.
